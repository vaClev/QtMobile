## 💻 Краткие пояснения к установке на Windows

### 1. Java JDK 17 - Требуется для сборки Android-приложений
https://adoptium.net/temurin/releases?version=17&os=any&arch=any
- Запустите скачанный `.msi`, следуйте инструкциям.
- Запомните путь установки (по умолчанию `C:\Program Files\Eclipse Adoptium\jdk-17...`).

### 2. Qt 6.8.3
- Запустите онлайн-установщик, войдите в Qt Account (бесплатно).
- На странице выбора компонентов раскройте `Qt` → `Qt 6.8.3`, отметьте:
  - `MinGW 64-bit`
  - `Android`
  - `Additional Libraries` → `Qt Multimedia`
- Нажмите **«Далее»**, дождитесь загрузки и установки.

### 3. Android SDK / NDK через Qt Creator
- Откройте Qt Creator.
- Перейдите `Правка` → `Параметры` → `Устройства` → `Android`.
- В поле **«Размещение SDK для Android»** укажите путь (например, `C:\Android`).
- Нажмите кнопку **«Установить SDK»** (она появляется, если папка пустая). Отметьте:
  - `Android SDK Platform 35`
  - `Command-line Tools`
  - `Platform-tools`
  - `Build-tools`
  - `NDK 26.1.10909125`
- Нажмите **«Установить»**.
- После завершения все пункты в настройках должны стать зелёными.
- Включите опцию **«Автоматически создавать комплекты для инструментария Android»**.

## 🚀 После установки
- Создайте новый проект `Qt Quick Application (CMake)`.
- На вкладке **«Проекты»** выберите Android-кит (например, `Android Qt 6.8.3 Clang x86_64` для эмулятора или `arm64-v8a` для телефона).
- Запустите сборку — если всё настроено правильно, приложение запустится на эмуляторе или устройстве.
